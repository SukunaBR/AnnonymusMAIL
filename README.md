## INSTALL: 

FUNCIONAMENTO EM LINUX E TERMUX SÃO IGUAIS

# No diretório onde queremos clonar o git:
$ git clone https://github.com/SukunaBR/AnnonymusMAIL.git

# Insira o diretório:
$ cd AnnonymusMAIL

# damos permissões de execução:
$ chmod + x mail.sh

# Finalmente execute o script:

# Linux
$ bash mail.sh

# Termux
$ ./mail.sh
